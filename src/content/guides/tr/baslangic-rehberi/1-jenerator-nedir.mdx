## Jeneratör Nedir?

Jeneratör, mekanik enerjiyi elektrik enerjisine dönüştüren bir cihazdır...

### Temel Bileşenler

1. **Motor**: Yakıt enerjisini mekanik enerjiye dönüştürür
2. **Alternatör**: Mekanik enerjiyi elektriğe çevirir
3. **Yakıt Sistemi**: Motora düzenli yakıt sağlar

> **Önemli Not:** Jeneratörlerin düzenli bakımı ömrünü uzatır.

```javascript
// Güç hesaplama örneği
const requiredPower = totalLoad * 1.25; // %25 yedek güç
console.log(`Gerekli jeneratör gücü: ${requiredPower} kW`);
```
